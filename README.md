# hide-n-seek
🕵️‍♂️ Offline face privacy tool - Auto-mask or warn when exposed. Built with Python/OpenCV.
# Hide&Seek 👤➡️🖤  
*"See without being seen" - A privacy shield for ethical hackers*  



## 🚀 Features  
- Real-time face detection & redaction  
- Visual/Audio warnings when unmasked  
- 100% offline processing (no telemetry)  
- Tkinter GUI for easy control  
- Configurable sensitivity
